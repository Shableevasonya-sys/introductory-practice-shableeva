# Примеры синтаксиса Markdown

## 1. Заголовки

# Заголовок 1 уровня
## Заголовок 2 уровня
### Заголовок 3 уровня
#### Заголовок 4 уровня

---

## 2. Форматирование текста

Обычный текст, а вот так можно сделать **жирный текст**, *курсив*, ~~зачёркнутый текст~~ и встроенный `код`.

Можно комбинировать: **жирный и _курсив_ одновременно**.

---

## 3. Списки

### Ненумерованный список

- Первый пункт
- Второй пункт
  - Вложенный подпункт
  - Ещё один подпункт

### Нумерованный список

1. Пункт номер один
2. Пункт номер два
3. Пункт номер три

## 5. Цитаты

> Это пример обычной цитаты.
> Цитату можно переносить на несколько строк, и она по-прежнему остаётся цитатой.

Можно вкладывать цитаты:

> Внешняя цитата
>> Внутренняя цитата

## 6. Блоки кода

Код внутри строки: `git status`.

Многострочный блок кода с указанием языка:

```bash
git status
git add .
git commit -m "Example commit"
git push
```

Ещё пример (Python):

```python
def hello(name: str) -> None:
    print(f"Hello, {name}!")

hello("World")
```
